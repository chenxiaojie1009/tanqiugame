接球小游戏
项目简介
一款简易休闲接球小游戏，基于前端技术开发，操作简单、玩法解压。玩家通过移动底部接球板，接住上方掉落的小球，接住得分、漏接扣分或结束游戏，适合新手练手、网页小游戏入门学习。
功能特性
简易流畅的小球自由下落物理逻辑
鼠标 / 键盘左右控制接球板移动
实时计分、分数记录功能
游戏开始、重新开始、游戏结束提示
适配电脑浏览器，界面简洁清爽
可自定义速度、小球数量、接球板大小
技术栈
前端：HTML5 + CSS3 + JavaScript
可选扩展：Canvas 绘图、动画帧 requestAnimationFrame

游戏操作说明
键盘：← → 方向键 控制接球板左右移动
鼠标：鼠标左右滑动控制接球板
目标：接住掉落的小球，避免小球落地
规则：接住小球加分，掉落小球扣除机会 / 直接结束游戏
自定义配置
可在 script.js 中修改以下参数，自由调整游戏难度：
小球下落速度
接球板宽度
小球生成频率
初始生命值 / 机会数
每次接球得分分值
